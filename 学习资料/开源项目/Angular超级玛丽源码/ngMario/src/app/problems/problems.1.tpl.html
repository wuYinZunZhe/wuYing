How to structure an app?

There's no official guidelines on many aspects like app structure

  • use Todd Motto styleguide
  • use boilerplates like ngBoilerplate
  • or use Yeoman generators like ngTailor

Boilerplates' benefits

  • Introduction to Grunt/Gulp, Bower, Karma runner
  • AngularJS specific tools like htm2js, ngAnnotate, UI Router, Restangular
  • business-oriented file structure
